# 发布日志

## [v0.4] - 2026-07-02

### 变更
- 按钮尺寸调整为 0.45rem 0.9rem，增加按下缩放反馈
- 导航栏添加当前页面激活态高亮
- 新增 `.page-head` / `.page-head-actions` 统一页面标题栏
- 标签管理页改用 `.tag-row` 轻量行替代 `<article>`
- 生成页左右栏均用 `<article>` 包裹，风格一致
- 模板选择项添加选中态高亮
- 卡片详情页改用 `white-space:pre-wrap` 替代 `replace('\n', '<br>') | safe`
- 新增 `.card-preview`、`.muted`、`.small`、`.empty-state` 等辅助类
- 整体间距、边框、阴影精调

---

## [v0.3] - 2026-07-02

### 变更
- 界面风格迁移到 Pico CSS，整体紧凑化
- 替换手写 CSS 为 Pico CSS v1 + 少量自定义覆盖
- app.css 重写，覆盖间距/字号/按钮/表单等变量
- 生成页改为双栏布局，用 `.generate-grid` + `.template-radio-item`
- 卡片列表、标签管理、模板管理页面均适配 Pico 语义结构

---

## [v0.2] - 2026-07-02

### 新增
- 标签独立管理（`/tags/`），支持创建、重命名、删除标签
- 标签与卡片改为多对多关系，卡片表单以复选框选择标签
- 导航栏新增「标签管理」入口

### 修复
- 卡片内容乱码问题（确保 UTF-8 编码流正确）

---

## [v0.1] - 2026-07-02

### 新增
- 项目骨架：Flask + SQLite + Jinja2
- 卡片 CRUD（标题、内容、标签、来源）
- 输出模板系统（笔记/时间线/对照），热加载
- 网页生成预览 + 导出为静态目录
- JSON 导入/导出
- 站点 Logo（SVG）
