# 变更日志

## v0.2.0 (2026-07-08)

### 新增

- 标签背景色：标签支持自定义背景色，卡片列表/生成网页同步展示
- 生成模板标签着色：导出的 HTML 标签使用实际背景色渲染

### 修复

- 生成页面标签筛选与显示适配标签对象格式
- 启动脚本 dev.bat：先检测端口占用并 kill，再启动前后端

## v0.1.0 (2026-07-03)

### 新增

- 项目初始化：Spring Boot 3.3 + Vue 3 + SQLite3 骨架搭建
- 卡片管理：卡片 CRUD、搜索筛选、标签关联、JSON 导入导出
- 标签管理：标签增删改、卡片计数统计
- 生成网页：3 种内置模板（笔记/时间线/网格对照）、预览与导出
- 用户认证：单用户注册/登录/JWT 鉴权/退出，BCrypt 加密
- 布局：左侧深色菜单 + 右侧内容区 + 顶部面包屑导航与用户信息
- 卡片列表：统计指标区域（卡片总数/标签总数/有来源/有标签）、表格排序、分页切换
- 卡片查看：阅读格式弹窗展示详情
- 固定布局：顶部和左侧菜单滚动时固定不动
- 时间格式：表格中时间显示优化为可读格式
- 开发脚本：dev.bat 一键启动、Vite HMR + Spring Boot DevTools 热加载
